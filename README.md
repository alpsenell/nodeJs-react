# Node js & React Todo

**1.** Clone the repository \
`git clone https://github.com/alpsenell/nodeJs-react.git`

**2.** Install dependencies \
`.npm i`

**3.** Start node server \
`.npm run start`

**4.** Start application \
`npm run dev`

**5.** Go to `localhost:5454` to see the application
